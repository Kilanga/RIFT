# RIFT — Scripts Vidéo (Storyboards Prêts à Filmer)

## VIDEO 1: TRAILER LANCEMENT (30 secondes)

**Objectif:** Faire connaître le jeu en 30 sec sans forcer
**Plateforme:** YouTube, TikTok, Reels, Shorts
**Tone:** Dynamique, hypnotisant, pas trop "pub"

### STORYBOARD DÉTAILLÉ

```
[0-2 sec] — FADE IN + HOOK
Video: Un boss moment spectaculaire (attaque épique du boss)
Audio: "DUN DUN" (son percussif)
Text overlay: "RIFT" (gros, néon)
Visual: Titre hypnotisant qui pulse

[2-5 sec] — GAMEPLAY MONTAGE
Video: Clips rapides combat (3 sec par clip, cuts agressifs)
  • Personnage attaque
  • Talont unlock transition
  • Boss hit épique
Audio: Beat de transition (drop électro light)
Text overlay: "Un roguelite nerveux"

[5-8 sec] — PROGRESSION HOOK
Video: Écran talent tree qui se remplit / classe unlock animation
Audio: "Sound design satisfying" (ding! pop! whoosh)
Text overlay: "Chaque run compte"
Visual: Couleurs néon flashy

[8-12 sec] — CLASS SHOWCASE (2 secondes par classe)
Video: 4 snippets rapides de combat différents (un par classe)
Audio: Beats électro continu (le même du début)
Text overlay: Pour chaque classe: "[Classe] — [1 mot key]"
  ex: "WARRIOR — Power"
      "MAGE — Speed"

[12-15 sec] — LEADERBOARD / COMPETITION
Video: Leaderboard zoom-in sur top 3 scores
Audio: Boss music (dramatic)
Text overlay: "Sois le meilleur" + emoji 🏆

[15-18 sec] — VISUAL FLEX
Video: Écran d'accueil / boss visual épique / couleurs néon
Audio: Music continues
Text overlay: "Techniquement beauté"

[18-28 sec] — GAMEPLAY FINAL (long clip cohérent)
Video: Une run "idéale" avec storyline (10 sec)
  1. Démarrage calm (1 sec)
  2. 1er combat rapide (3 sec)
  3. Talent unlock (1 sec)
  4. Boss fight épique (4 sec)
  5. Victory screen (1 sec)
Audio: Music peak + sound design épique
Text overlay: rien, laisser le gameplay parler

[28-30 sec] — CTA FINAL
Video: Logo du jeu + Download button visual (mockup Google Play)
Audio: Final "drop" électro
Text overlay: 
  "RIFT"
  "Disponible maintenant"
  "Android"
  "Lien en bio 👇"
```

### TECHNICAL SPECS
- **Format:** 9:16 (vertical pour TikTok/Reels/Shorts, adapte aussi en 16:9 pour YouTube)
- **FPS:** 30fps ou 60fps
- **Bitrate:** 1080p min
- **Audio:** Mono ou Stéréo (bien audible sur téléphone)
- **Music:** Royalty-free électro/synthwave (Epidemic Sound free trial, ou libre de droit)

## HOW TO PRODUCE THESE FAST

### Option 1: Manual Recording (You)
1. Record gameplay on phone/emulator = MP4 files
2. Import in **CapCut** (free)
3. Add music, transitions, text overlays
4. Export as MP4
**Time:** ~1 hour per video | **Quality:** Real gameplay

### Option 2: AI Video Generation (Faster, Less Quality)
Services like **Synthesia**, **D-ID**, **Pictory** can auto-generate videos from scripts.
But they **won't look like real RIFT gameplay** — trade-off.

### Option 3: Hybrid (RECOMMENDED)
1. Record 5-10 min of actual gameplay (best parts)
2. Use CapCut to edit into shorts
3. Add AI voiceover if needed (natural sounding)
**Time:** ~30 min per batch | **Quality:** Good balance

### MUSIC SOURCES (Free & Royalty-Free)
- YouTube Audio Library (free, no copyright claim)
- Epidemic Sound (free trial)
- Pixabay Music (free)
- FreePM (free)
- Incompetech (free)
