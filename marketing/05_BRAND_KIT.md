## 🔤 TYPOGRAPHIE

### Font Recommendations
- **Titles/Headers:** Futura, Montserrat Bold, ou Orbitron (futuriste)
- **Body text:** Inter, Roboto, ou OpenSans (lisible)
- **Accent/Logo:** Space Mono or IBM Plex Mono (tech feel)

### Sizing
- **Main title:** 48px+ (bold)
- **Subtitle:** 32px (semi-bold)
- **Body:** 14-16px (regular)
- **Small text:** 11-12px (muted)

### Rules
- ALL CAPS pour titres importants (ex: "RIFT", "AVAILABLE NOW")
- Lowercase pour body (exception: names/proper nouns)
- Letterspacing: +2px pour titres, 0 pour body
- Line height: 1.6 pour lisibilité mobile

## 🎬 VIDEO STYLE GUIDE

### Editing
- **Cut frequency:** Tous les 0.5-1 sec (nerveux, rapide)
- **Transitions:** Simple fade ou cut (pas de effects flous)
- **Speed:** 1x pour narrative, 1.5x-2x pour montage intense
- **Slow-mo:** Pour boss moments épiques (0.75x ou 0.5x)

### Color Grading
- **Saturation:** +20% (couleurs un peu boostées)
- **Contrast:** +15% (lisibilité)
- **Exposure:** Normal ou -5% (image plus "cool")
- **Temperature:** Slightly cooler (blueish tint, tech feel)

### Text Overlays
- **Font:** Montserrat Bold ou Space Mono
- **Size:** 48px+ pour titles, 24-32px pour body
- **Placement:** Top-third ou bottom-third (safe margins)
- **Durée affichage:** Min 2 sec (readability), max 5 sec (pas trop long)
- **Drop shadow:** Oui, pour contraste (shadow: 0 0 8px rgba(0,0,0,0.8))
- **Color:** Blanc (#FFF) ou néon (#FF00FF) sur fond foncé

### Audio
- **Music style:** Synthwave, electronic, arcade-inspired
- **BPM:** 90-140 (energetic but not chaotic)
- **SFX:** Retro arcade sounds (ding, pop, whoosh, power-up) + game audio
- **Voice:** Si présent, voix neutre pro française ou anglaise (pas excité)
- **Levels:** Music -18db, SFX -12db, voix -8db

## 📱 SOCIAL MEDIA POST GUIDE

### TikTok / Instagram Reels / YouTube Shorts
**Format:** 9:16 (Portrait)
**Duration:** 15-30 sec max
**Aspect ratio:** Vertical avec safe area 20px margins
**Text size:** Min 28px (readability from thumbs)
**Hook:** 3 sec max

### Twitter/X
**Text length:** 280 char max
**Images:** 16:9 or 1:1 (test both)
**Hashtags:** 3-5 (not spammy)
**Links:** Bit.ly shortened (track clicks)

### Reddit
**Title:** 5-10 words, not clickbaity
**Body:** Authentic, conversational
**Images:** 16:9 or 4:3 (Reddit compresses)
**Spacing:** Use line breaks (readability)

### Discord
**Embeds:** Color #FF00FF (Magenta)
**Roles:** role-marketing for announcements
**Tone:** Friendly, casual dev voice

## ✍️ VOICE & TONE

### Core Message
"RIFT is fast, intense, and visually stunning. Every run matters. Join the community and find your limit."

### Tone Characteristics
- **Energetic but authentic** (no BS marketing, real dev talk)
- **Playful with skill hints** (tips feel like helping, not showing off)
- **Inclusive** (all skill levels welcome)
- **Respectful of player time** (short runs, no grind pressure)

### Language Examples

**What to say:**
- "Your best run yet?"
- "What's your strategy?"
- "Just dropped a new build guide"
- "Thanks for the feedback, adding it to the roadmap"
- "Breaking your own record feels good"

**What NOT to say:**
- "This game is the best" (subjective)
- "You MUST play this" (pushy)
- "Grind for hours" (misleading)
- "Pay to win" (not true, but avoid even mentioning)

## 📊 CHANNEL-SPECIFIC RULES

### TikTok
- Use trending sounds (check weekly)
- Post 3-5x per week
- Hashtag strategy: #FYP #ForYou + niche + game-specific
- Engage with comments FAST (within 1 hour)
- Collab with small gaming creators if possible

### Instagram Reels
- Post 2-3x per week
- Hashtags: 20-30 relevant ones (auto-discovery)
- Stories: Teases between posts
- Captions: Casual, question at end (engagement)
- Reply to DMs and comments
- Use Stickers: polls, questions, links

### YouTube Shorts
- Post 1-2x per week
- Always link to full channel (if later have longer vids)
- Shorts don't need title, description does
- Hashtags in description work (#shorts #gaming)

### Reddit
- Authentic only (no "PROMOTE" energy)
- Engage with comments like a real community member
- Cross-post relevant advice/tips
- Avoid consecutive posts same subreddit (spread over days)
- Award comments you like (builds goodwill)

### Twitter/X
- Tweet 1-2x per day (not overwhelming)
- Reply to game dev conversations
- Share wins/community highlights
- Retweet fan content
- Keep threads under 5 tweets

### Discord
- Messages casual (not corporate)
- React fast to new member intros
- Monthly community event (leaderboard, challenge, AMA)
- Pin important announcements
- Use roles for organization of info
